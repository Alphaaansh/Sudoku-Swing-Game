# Sudoku Swing Game

A fully functional Sudoku puzzle game built using **Java Swing**

---

## Features

-  Random puzzle every time
-  Mouse-based input
-  Sudoku grid with 3x3 blocks
-  Built with Java 23 + Swing
-  Runs fully offline

---

## Technologies Used

- Java 23
- Java Swing (UI framework)
- IntelliJ IDEA
